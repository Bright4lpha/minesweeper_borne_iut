# minesweeper_borne_iut

A minesweeper game for the arcade of the IUTLCO. Project of 3rd year of BUT Info.

## Commands

Joysticks :

-   up : arrow up
-   down : arrow down
-   left : arrow left
-   right : arrow right

Buttons:

-   Top :

    -   1st : X - Back/Exit
    -   2nd : Y - (Difficulty ? / Score)
    -   3rd : Z

-   Bottom :
    -   1st : A - Button to dig
    -   2nd : B - Button to flag
    -   3rd : C - Dig/Flag - Restart
